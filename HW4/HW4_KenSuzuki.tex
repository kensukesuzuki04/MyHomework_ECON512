\section*{Problem 1: Dart-throwing method using Quasi-Monte Carlo}

First define the function \code{pi\_ind.m} which is the indicator function $\mathbbm{1}\lmprn{x^2+y^2\leq 1}$.

\lstinputlisting{pi_ind.m}

Using \code{haltonseq()} that is provided in the class, I draw 10,000 two-dimensional random numbers between 0 and 1. Using the quasi-Monte Carlo approach, I approximate the integration as:

\begin{align}
\int_0^1 \int_0^1 \mathbbm{1}\lmprn{x^2+y^2\leq 1}dydx \approx \frac{(1-0)(1-0)}{10000}\sum_{j=1}^{10000}\mathbbm{1}\lmprn{x_j^2+y_j^2\leq 1}
\end{align}

where $x_j$ and $y_j$ is the sequence of random draws. Our result is:

\begin{align*}
\pi \approx 3.1448
\end{align*}

\section*{Problem 2: Dart-throwing method using Newton-Cotes}

We again use the function \code{pi\_ind.m} and employ the Newton-Cotes approach. First define the width of partition $h$ as

\begin{align*}
h = \cf{1-0}{10000}
\end{align*}

where 10,000 is same as the number of draws we used in problem 1. Using this partition, we creates the sequences of $x$ and $y$ as

\begin{align*}
&x_j = 0 + (j - 1/2)\times h \\
&y_j = 0 + (j - 1/2)\times h 
\end{align*}

for $j=1,...,10000$. We use the equal weight  $w_j = h$ for numerical integration. I first integrate over $y$ for given $x_k$ as follows:

\begin{align}
f(x_k) = \int_{0}^1 \mathbbm{1}\lmprn{x_k^2+y^2\leq 1}dy \approx h \times \sum_{j=1}^{10000} \mathbbm{1}\lmprn{x_k^2+y_j^2\leq 1} \equiv \tilde{f}(x_k)
\end{align}

Then we integrate over $x$ as:

\begin{align}
\int_0^1 f(x_k) d x_k \approx h \times \sum_{k=1}^{10000} \tilde{f}(x_k)
\end{align}

The result is:

\begin{align*}
\pi \approx 3.1416
\end{align*}

\section*{Problem 3: Pythagorean formula with Quasi-Monte Carlo}

Define the function \code{pi\_root.m} which is returns $\sqrt{1-x^2}$. Now we only need to draw random numbers for $x$. We again use \code{haltonseq()} to draw 10000 random numbers. Numerical integration is given by

\begin{align}
\int_0^1 \sqrt{1-x^2} dx \approx \cf{1-0}{10000} \sum_{j=1}^{10000}\sqrt{1-x_j^2}
\end{align}

The result is

\begin{align*}
\pi = 3.1422
\end{align*}


%%%%%%%%%%%%%%%%%%%%%%
\section*{Problem 4: Pythagorean formula with Newton-Cotes}

We use the same width of partition $h$ as in the problem 2 and create the sequence of $x$ analogously. Using the equal weight $w_j = h$, numerical integration is given by

\begin{align*}
\int_0^1 \sqrt{1-x^2} dx \approx h \times \sum_{j=1}^{1000}\sqrt{1-x_j^2}
\end{align*}

The result is

\begin{align*}
\pi = 3.1416
\end{align*}

\section*{Problem 5: Comparison}

For the pseudo-Monte Carlo method, we use \code{rand()} to generate random numbers. We simulate 200 times to compute the mean squared error. We compute the mean squared errors for different number of draws; 100, 1,000 and 10,000 times. Errors are computed by taking difference between the numerically computed values and the real $\pi$ value (computed using matlab \code{pi} command). For Newton-Cotes method, we employ the same method as we have outlined above. We compute with three different number of nodes. Results are presented below:


\begin{table}[h]
\begin{center}
\begin{tabular}{|c|r|r|r|}
\hline
                          & \multicolumn{1}{c|}{100} & \multicolumn{1}{c|}{1,000} & \multicolumn{1}{c|}{10,000} \\ \hline
Dart-throwing, Pseudo-Monte Carlo (MSE)   
&  0.0246    & 0.0025     & 0.0003      \\ \hline
Dart-throwing, Newton-Cotes (Sq'd error)
&  0.0075    & 0.0009     & 0.0001      \\ \hline
Pythagorean, Pseudo-Monte Carlo (MSE)   
&  $1.0e-0.5\times 0.1458$    & $1.0e-0.5\times 0.0007$     & $1.0e-0.5\times 0.0000$      \\ \hline
Pythagorean, Newton-Cotes (Sq'd error)
&  $1.0e-0.6\times 0.1185$    & $1.0e-0.6\times 0.0001$     & $1.0e-0.6\times 0.0000$      \\ \hline
\end{tabular}
\end{center}
\end{table}

(Mean) squared errors are smaller in Newton-Cotes method than in Pseudo-Monte Carlo method. Comparing dart-throwing to Pythagorean approach, the latter works better because it relies on continuous function to be integrated.
